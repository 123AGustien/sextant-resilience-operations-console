## 🧠 System Overview

The Sextant Control Room is a deterministic, graph-based simulation platform designed to model:

- systemic risk propagation  
- dependency failure cascades  
- operational resilience dynamics  
- cross-domain interaction effects  

All outputs are synthetic and used for simulation and research purposes only.

---

## ⚠ System Classification

- Simulation-only environment  
- No real financial transactions  
- No live banking or user data  
- No external system integration  
- Deterministic synthetic outputs only  

---

## 🧠 Core Engine (RP-04)

The system is powered by a deterministic cascade engine:

**RP-04 Responsibilities**
- Dependency graph execution  
- Cascade propagation logic  
- Severity state transitions (S1–S5)  
- Reproducible simulation outputs

## 🌊 Simulation Flow

Dependency → Stress → Failure → Cascade → Observability → Recovery → Stabilisation

Key characteristics:
- weighted dependency propagation  
- cross-domain interaction effects  
- delayed failure detection  
- systemic risk clustering behaviour

## 📊 Risk Model

**RI = (DS × PS × CD) / OD**

Where:
- DS = Dependency Strength  
- PS = Propagation Sensitivity  
- CD = Coupling Density  
- OD = Observability Delay
