# 🧪 Stress Model

Each subsystem uses normalized values:

- 0.00–0.20 Stable
- 0.21–0.40 Watch
- 0.41–0.60 Stress
- 0.61–0.80 Severe Stress
- 0.81–1.00 Crisis
