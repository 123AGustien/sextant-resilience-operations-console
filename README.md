## 🎯 System Modes

Simulates healthy infrastructure state

### 🟡 Node Failure
Simulates single-node degradation and recovery response

### 🔴 Cascade Failure
Simulates multi-node cascading system failure

## 🧠 Deterministic Engine

- Same input → same output
